# Azure Sentinel Training Lab

## Overview
This directory contains the official Azure Sentinel Training Lab from Microsoft's GitHub repository. This comprehensive training provides hands-on experience with Microsoft Sentinel setup, configuration, and operations.

## Source
Based on: [Azure Sentinel Training Lab](https://github.com/Azure/Azure-Sentinel/blob/master/Solutions/Training/Azure-Sentinel-Training-Lab/Modules/Module-1-Setting-up-the-environment.md)

## Lab Structure

### Module 1: Setting up the Environment
- Environment setup and prerequisites
- Azure Sentinel workspace deployment
- Initial configuration

### Module 2: Data Connectors
- Connecting various data sources
- Configuring data ingestion
- Validating data flow

### Module 3: Analytics Rules
- Creating detection rules
- Tuning and optimization
- Testing and validation

### Module 4: Incident Management
- Incident response workflow
- Investigation techniques
- Case management

### Module 5: Hunting
- Proactive threat hunting
- Advanced KQL queries
- Custom hunting rules

### Module 6: Watchlists
- Creating and managing watchlists
- Data enrichment
- Threat intelligence integration

### Module 7: Threat Intelligence
- Threat intelligence platforms
- IOC management
- Threat feeds integration

### Module 8: Sentinel Solutions
- Solution deployment
- Content hub management
- Advanced configurations

## Getting Started

1. Review the prerequisites in Module 1
2. Follow the setup instructions step by step
3. Complete each module in sequence
4. Document your progress and findings
